# SafeExif

SafeExif is a local-first Android app that allows users to inspect and safely remove metadata from their photos and videos. Its mission is to empower users with control over the information embedded in their files, without collecting or transmitting any personal data.

## Purpose

Files often carry hidden metadata — timestamps, location, device details, and software information — that can reveal more than users intend. SafeExif provides transparency and control, allowing users to:

- See exactly what metadata exists in each file
- Remove sensitive metadata while preserving the file content
- Make informed decisions about sharing or storing files

## Key Features

- Human-readable metadata reports for photos and videos
- Safe metadata removal (“scrubbing”) that preserves visual content
- Local-first processing: no uploads, no analytics, no third-party tracking
- Future-ready: designed for potential expansion into advanced metadata inspection and security tools

## Installation & Usage

- Android app coming soon on Google Play Store  
- Files are selected manually via the interface, ensuring user control
- Metadata reports provide clear summaries and risk flags

## Philosophy

SafeExif operates under the principles of **agency-centered security**: empowering users, preserving privacy, and promoting clarity over alarmism. This project is a first step in a family of tools designed to enhance user knowledge and control of file data.
